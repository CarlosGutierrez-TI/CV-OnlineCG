# Comandos Básicos de Git - Ejercicio Práctico

A continuación se define y ejemplifica el uso de los comandos básicos solicitados:

* **`git --version`**: Muestra la versión instalada de Git en el sistema operativo.
  * *Ejemplo de uso:* `git --version`
* **`git init`**: Inicializa un repositorio de Git en la carpeta actual, creando el entorno de seguimiento (directorio oculto `.git`).
  * *Ejemplo de uso:* `git init`
* **`git add`**: Agrega los cambios del directorio de trabajo al área de preparación (staging).
  * *Ejemplo de uso:* `git add .`
* **`git commit`**: Registra los cambios del área de preparación en el historial del repositorio.
  * *Ejemplo de uso:* `git commit -m "Primer commit"`
* **`git status`**: Muestra el estado actual del árbol de trabajo y del área de preparación.
  * *Ejemplo de uso:* `git status`
